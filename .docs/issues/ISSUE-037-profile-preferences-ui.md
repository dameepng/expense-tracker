# ISSUE-037: Profile - Preferences & Settings Menu UI

## Deskripsi
Menambahkan bagian menu "Preferensi" di dalam halaman Profile. Menu ini menggunakan *list item* interaktif yang dikelompokkan ke dalam sebuah Card (rounded) bergaya Material 3. Bagian ini fokus pada UI (Statik mockup).

## Acceptance Criteria
- [ ] Membuat komponen *reusable* `SettingsGroup` (kotak penampung list menu) dan `SettingsItem` (baris menu dengan leading ikon, teks judul, dan trailing icon/chevron).
- [ ] Menambahkan menu "Tema Aplikasi" (dengan indikasi teks Dark/Light).
- [ ] Menambahkan menu "Mata Uang Utama" (dengan indikasi teks IDR).
- [ ] Menambahkan menu "Bahasa" (dengan indikasi teks Indonesia).
- [ ] Menggunakan Material 3 Colors (`surfaceVariant` atau similar) untuk background grup menu.

## Skills
- Pembuatan Reusable UI Component di Compose.
- Material 3 List & Card Design Pattern.
