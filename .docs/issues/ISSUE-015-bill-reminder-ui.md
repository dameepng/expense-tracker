# ISSUE-015: Bill Reminder List & Form Screen

## Deskripsi
Membuat UI untuk menampilkan daftar tagihan dan form tambah reminder baru. Navigasi dari ikon 🔔 di Home.

## Acceptance Criteria
- [ ] `ReminderListScreen` menampilkan daftar tagihan aktif dengan swipe-to-delete
- [ ] `ReminderFormScreen` dengan input: nama, nominal, tanggal deadline, pilih wallet
- [ ] `ReminderViewModel` dan `ReminderFormViewModel`
- [ ] Navigasi: Home 🔔 → ReminderList → FAB → ReminderForm
- [ ] Unit test: ViewModel logic
- [ ] CI GitHub Actions pipeline hijau
